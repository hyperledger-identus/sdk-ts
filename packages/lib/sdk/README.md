You can install the SDK with npm, yarn, etc.

```sh
npm i @hyperledger/identus-sdk --save

```

Or using yarn:
```sh
yarn add @hyperledger/identus-sdk
```

Using the library:

```javascript
import { Apollo } from '@hyperledger/identus-sdk';

const apollo = new Apollo();
```
